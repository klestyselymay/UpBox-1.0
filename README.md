# UpBox-1.0
UpBox-1.0 is a hacking app that also allow you to chat with your team while hacking.

HOW TO INSTALL:

step 1: 
```
git clone
```
step 2:
```
pip install -r requirements.txt
```

HOW TO USE:
```
python UpBox-1.0.py
```

to message 
```
hello world!
```

to enter a commnad use
```
/
```

first command:
```
/test
```
